About The A* Algorithm

What is the A* Algorithm?

The A* algorithm is an algorithm that finds the shortest path between two points on a graph. A* is considered to be the optimal pathfinding algorithm because it searches for the shorter paths first. It is able to do this because the algorithm is an Informed Search Algorithm, meaning it takes advantage of a heuristic function to make informed decisions throughout.

A heuristic function is a function that ranks potential branches in order to decide which branch to take next. In the case of the visualizer, since our graph is on a globe, our heuristic function is the distance between the current city and the destination across the Earth's surface. In A*, the heuristic function is compared with the cost function in order to make its next decisions.

The general equation is: f(current_node) = g(current_node) + h(current_node).

It is crucial that the heuristic value never exceeds the actual cost to ensure the accuracy of the algorithm. If the heuristic is not properly bounded, the algorithm may overestimate or underestimate the true cost, leading to suboptimal paths.

General Algorithm

Create a Map to hold the path from the start to the destination.
Place starting_node into the open_queue.
While the open_queue is not empty:

  • Pop the lowest value from open_queue and set it to current_node:
    • In my algorithm, I used a min priority queue to easily access the lowest value.
    • Let f(current_node) = g(current_node) + h(current_node).
  • If current_node is destination_node, we have found the solution.
    • Reconstruct the path backwards and return the path.
  • For every neighbor of current_node:
    • Set neighbor_cost = g(current_node) + w(current_node, neighbor).
    • If neighbor is in open_queue:
      • If g(neighbor) ≤ neighbor_best_cost, continue.
    • Else if neighbor has been visited:
      • If g(neighbor) ≤ neighbor_best_cost, continue.
      • Else:
        • Add neighbor to the open_queue.
        • Set h(neighbor) to be the heuristic distance to destination_node.
    • Set g(neighbor) = neighbor_best_cost.
    • Set the path mapping of neighbor as current_node.
  • Add current_node to the visited set.

If current_node never reaches the goal, return an error.
